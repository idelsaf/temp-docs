# Working with Tasks

This guide covers all the essential task management features in TaskTracker.

## Task Properties

Each task in TaskTracker contains the following properties:

| Property | Description | Required |
|----------|-------------|----------|
| Title | A brief summary of the task | Yes |
| Description | Detailed information about the task | No |
| Assignee | The team member responsible | Yes |
| Priority | Low, Medium, High, or Urgent | No |
| Due Date | When the task should be completed | No |
| Tags | Labels for categorization | No |

## Creating Tasks

To create a new task:

1. Click **New Task** in the navigation bar
2. Fill in the task details
3. Set the priority and due date (optional)
4. Click **Create**

## Editing Tasks

You can modify any task at any time:

1. Click on the task to open it
2. Click the **Edit** button
3. Make your changes
4. Click **Save**

## Task Statuses

Tasks progress through the following statuses:

- **To Do**: New tasks waiting to be started
- **In Progress**: Tasks currently being worked on
- **Review**: Tasks awaiting review or approval
- **Done**: Completed tasks

## Deleting Tasks

!!! danger "Warning"
    Deleted tasks cannot be recovered. Proceed with caution.

To delete a task:

1. Open the task
2. Click the **Delete** button
3. Confirm the deletion
